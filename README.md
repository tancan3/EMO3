# 心愈 AI - 心理健康智能助手

一个基于人工智能的心理健康评估与陪伴系统，采用 Flask + Vue 技术栈构建。

---

## 一、功能模块

### 1. 用户系统

| 功能 | 路由 | 说明 |
|------|------|------|
| 用户注册 | `/register` | 用户名密码注册 |
| 用户登录 | `/login` | 登录后保存 session |
| 用户退出 | `/logout` | 清除 session |
| 游客模式 | - | 未登录用户可部分功能使用 |

### 2. 抑郁症心理测评

| 功能 | 路由 | 说明 |
|------|------|------|
| 测评首页 | `/` | 介绍页，点击开始测评 |
| 测评页面 | `/detect` | 问卷填写，支持音频上传 |
| 测评结果 | `/report` | 雷达图展示+风险评估 |
| 历史记录 | `/api/history` | 获取历史测评记录 |

**问卷维度：**
- PHQ-9 抑郁量表（9题）
- GAD-7 焦虑量表（7题）
- 睡眠质量（4题）
- 压力水平（4题）
- 社交能力（3题）
- 自我评价（3题）
- **语音情感识别**（Wav2Vec2 模型）

### 3. 情绪垃圾桶（AI对话）

| 功能 | 路由 | 说明 |
|------|------|------|
| 对话页面 | `/soothe` | AI 情绪安抚对话 |
| 聊天接口 | `/api/chat` | 调用 Qwen2.5 大模型 |
| 语音输入 | - | Web Speech API 语音识别 |

### 4. 每日打卡

| 功能 | 路由 | 说明 |
|------|------|------|
| 打卡页面 | `/profile` | 个人中心内 |
| 打卡接口 | `/api/checkin` | 5级情绪评分 + 笔记 |

### 5. 心理科普文章

| 功能 | 路由 | 说明 |
|------|------|------|
| 文章列表 | `/articles` | 分类展示科普文章 |
| 文章详情 | `/article/<id>` | 阅读文章内容 |
| 文章接口 | `/api/articles` | 获取文章列表 |

### 6. 互助社区

| 功能 | 路由 | 说明 |
|------|------|------|
| 社区首页 | `/community` | 帖子列表 |
| 发布帖子 | `/api/posts` | 匿名发布心情 |
| 点赞帖子 | `/api/posts/<id>/like` | 点赞/取消点赞 |

### 7. 个人中心（数据可视化）

| 功能 | 路由 | 说明 |
|------|------|------|
| 个人中心 | `/profile` | 数据仪表盘 |
| 统计数据 | `/api/stats` | 心理健康评分等 |
| 数据导出 | `/api/export` | CSV 导出 |

---

## 二、技术架构

### 后端
- **Flask** - Web 框架
- **SQLite** - 数据库
- **Transformers** - Hugging Face 模型库
- **Wav2Vec2** - 语音情感识别
- **Qwen2.5** - 大语言模型（情绪安抚对话）

### 前端
- **Tailwind CSS** - UI 样式
- **Chart.js** - 数据可视化
- **Web Speech API** - 语音识别

### 数据表
- `users` - 用户表
- `questions` - 测评问题表
- `records` - 测评记录表
- `daily_checkin` - 每日打卡表
- `articles` - 科普文章表
- `posts` - 社区帖子表
- `post_likes` - 帖子点赞表

## 五、快速开始

### 环境要求
- Python 3.10+
- CUDA (可选，用于加速模型推理)

### 安装依赖
```bash
pip install -r requirements.txt
```

### 环境配置（可选）
```bash
# 复制配置示例文件
copy .env.example .env

# 编辑 .env 文件配置您的参数
notepad .env
```

### 启动服务
```bash
python app.py
```

访问 http://localhost:5000
